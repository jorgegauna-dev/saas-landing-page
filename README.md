# SaaS Landing Page

Modern responsive SaaS landing page built with HTML and Tailwind CSS.

## Description
This project is a modern SaaS landing page designed for startups and software products.  
It includes common sections used in real business websites such as a hero section, feature highlights, pricing plans, testimonials, and a call-to-action area.

The goal of this project is to demonstrate front-end development skills, responsive layout design, and clean UI structure using Tailwind CSS.

## Features
- Hero section
- Features section
- Pricing plans
- Testimonials
- Call-to-action section
- Responsive design

## Technologies
- HTML
- Tailwind CSS
